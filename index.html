Minha Biblioteca

A Biblioteca Virtual é um site interativo desenvolvido com HTML, CSS e criado para facilitar o acesso à leitura de forma prática, organizada e envolvente. A plataforma reúne uma coleção de livros digitais separados por gêneros, com capas, descrições e links para leitura ou download. O layout responsivo garante uma boa experiência em qualquer dispositivo, e os recursos em JavaScript permitem buscar títulos, navegar por categorias e marcar livros como favoritos. Com um visual limpo e intuitivo, o site promove o hábito da leitura e torna o conhecimento acessível a todos, funcionando como uma estante digital sempre ao alcance do leitor.

criando uma aventura

"Criando uma Aventura" é um site interativo que leva o usuário a viver uma jornada única através de uma história fictícia criada especialmente para o ambiente digital. Desenvolvido com HTML, CSS , o projeto combina elementos narrativos, escolhas interativas e um design imersivo para transformar a leitura em uma verdadeira experiência de exploração. A narrativa se desenrola conforme as decisões do usuário, que influencia o rumo dos acontecimentos e descobre diferentes finais possíveis. Com um visual envolvente e responsivo, o site oferece uma forma moderna e criativa de contar histórias, conectando tecnologia e imaginação para despertar o senso de aventura em cada visitante. Ideal para leitores curiosos, fãs de RPGs e todos que amam se perder em boas histórias.

criando uma aventura

O projeto é uma aplicação web interativa desenvolvida com HTML, CSS e JavaScript, que convida o usuário a mergulhar em uma história original de aventura. O site apresenta uma narrativa dividida em capítulos ou seções, onde o visitante acompanha os acontecimentos por meio de texto, imagens e elementos visuais imersivos. A proposta é oferecer uma experiência envolvente, combinando leitura e exploração digital para dar vida à jornada fictícia criada pelo autor.
